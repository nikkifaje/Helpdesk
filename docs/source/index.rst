HelpDesk Team Development and Customization           				
============================================





Project Vision
---------------
**Develop a ticketing system that aids with organizing information, empowering the team, and maintaining force with in project development.**

Requirements
-------------

- Users can submit request at any time
- Requests can be tracked
- Admin user can view all requests
- Users can review all requests that they have submitted
- While request is still open, users can update requests details
- Request can be closed by both admin and end users
- Communication during project development can be saved with in Helpdesk
- Automated notifications
- Admin can communicate internally


Definition of Done (DoD) 
------------------------

- Help Desk is housed on ACU server
- Application has been tested by Admin and Department Users
- Help Desk is able to be accessed through ArrowNet
- Admin and User documentation is completed and linked to BI site
- Administrators Roles and Responsibilities outline on access set up
